## My Portfolio

This is my personal portfolio website, 
built to showcase my interests, skills, and ongoing learning journey at the intersection of biotechnology and programming.

---

## Overview

The website reflects my inclination toward understanding systems deeply, 
approaching problems analytically, and presenting ideas with clarity and structure. 
It combines a minimal design with subtle visual enhancements to create a clean and engaging interface.

---

## Features

* **Dynamic Gradient Header with Animation**
  The portfolio opens with a visually engaging heading that utilizes animated gradient text and glow effects, creating a strong first impression while reflecting a modern, design-conscious approach.

* **Structured Sectional Layout**
  The content is organized into clearly defined sections—**Stats, About, Interests, and Skills**—each designed to present information in a logical, readable, and aesthetically consistent manner.

* **Dedicated Contact Interface**
  A focused contact section provides direct access to my professional and social profiles, including GitHub, LinkedIn, and Instagram, ensuring easy connectivity and a tangible digital presence.

* **Interactive Skill Representation**
  Skills are displayed through responsive UI elements that incorporate hover effects and subtle transitions, enhancing user interaction while maintaining clarity.

* **Custom Styled Interest Modules**
  Interest areas are presented through visually distinct cards, each potentially featuring background imagery and layered styling to emphasize depth and individuality.

* **Consistent Visual Theme**
  The website maintains a cohesive design language through the use of gradients, neon accents, and dark-themed backgrounds, ensuring both visual appeal and readability.

* **Responsive and Lightweight Design**
  Built using fundamental web technologies (HTML & CSS), the site remains lightweight, fast-loading, and adaptable across different screen sizes.
